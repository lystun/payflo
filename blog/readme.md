# SPACEPADDY-API

> Backend API for the Spacepaddy Blog Service

## Usage

Rename the ".env.example" file to .env and update the 
values/configurations to your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in development mode
npm run dev

# Run in Production mode
npm start
```
